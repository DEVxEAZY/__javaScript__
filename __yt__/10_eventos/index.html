Eventos JavaScript mais comuns

  • onclick: Acionado quando o usuário clica em um elemento.
  • onmouseenter: Acionado quando o ponteiro do mouse entra na área de um elemento.
  • onmouseleave: Acionado quando o ponteiro do mouse sai da área de um elemento.
  • onchange: Acionado quando o valor de um elemento (como input ou select) é alterado.
  • onfocus: Acionado quando um elemento recebe o foco (ex: clicar dentro de um input).
  • onblur: Acionado quando um elemento perde o foco.
  • onkeydown: Acionado quando uma tecla é pressionada.
  • onkeyup: Acionado quando uma tecla é solta.
  • onload: Acionado quando a página ou um recurso termina de carregar completamente.
  • onsubmit: Acionado no momento em que um formulário é enviado.
